FUNÇÕES

nome=function(parametro){
   codigo
}
muito simples criar funções com parametros no R

Funções sem parametros tem um simples diferença
declare ela do mesmo jeito
mas sem parametros
no codigo voce deve definir uma variavel que receba oq o usuario digitar
nome=function(parametro){
   var=readline(prompt="Defina uma mensagem ao usuario")
   var=as.integer(var)
}
no exemplo acima definimos a variavel var
na primeira linha chamamos um prompt para que o usuario digite oq quiser
na segunda linha pegamos oq ele digitou (as) e transformamos no tipo de dado desejado (integer), após isso alocamos em alguma variavel (var)
dessa maneira podemos usar oq o usuario digitou em nosso codigo

Tabela
Objeto que contem multiplos vetores do mesmo tamanho
qualquer tipo de vetor
data.frame --> cria-se tabelas
nome=c("Eduardo", "Lucas", "Guilherme")
curso=c("Superior", "Médio", "Técnico")
idade=c(19, 20, 20)
salario=c(400, 2800, 1300)
sexo=c("M", "M", "M")
df=data.frame(nome, curso, idade, salario, sexo, stringsAsFactors = FALSE)
df

cria-se varios vetores
e depois so os junta na tabela
stringAsFactors = FALSE
representa que as strings não são fatores
mas apenas strings de caracteres msm

subset --> coluna e forma de subdivisão
cria uma subtabela com os dados selecionados
é como se vc desse um select no que vc quisesse do BD
subset(nometabela, campo=="parametro") #ele trara outra tabela, com os dados filtrados pelo campo selecionado

select --> retorna as colunas que satisfazem a subdivisão
subset(nometabela, select=-campo) #ele vai tirar o campo selecionado e retornar o resto
ou seja ele realiza o select, e retorna o resto
subset(nometabela, select=campo:campo) #os dois pontos indicam o intervalo
